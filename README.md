# Application de Gestion d’Agendas

## Description du sprint 1 : ajout agenda
L'intêrét de cette modification est de pouvoir permettre à un utilisateur d'ajouter un agenda en plus de son agenda par défaut. Il aura ensuite la possibilité d'ajouter des RDV en précisant quel sur quel agenda il sera. 
Finalement, sur son calendrier il pourra choisir quel(s) agenda(s) afficher ou non.
